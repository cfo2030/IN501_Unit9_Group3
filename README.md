# IN501 Unit 9 Assignment

## Description
This assignment transforms the development of a Convolutional Neural Network (CNN) for digit classification (using the MNIST dataset) into a collaborative group project. Each team will build a baseline model, then experiment with individual variations, hypothesize about outcomes, and finally present the results and lessons learned.

## Required Third-Party Libraries
- tensorflow
- matplotlib
- numpy

Command to install libraries via *requirements.txt*:  
```pip install -r requirements.txt```
